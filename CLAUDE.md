# Running Playbook

ランニングのトレーニング計画・大会計画・書籍からの知見・ツールをまとめた個人リポジトリ。

## ディレクトリ構成

- `training_plans/` — 大会に向けたトレーニング計画（大会日_種目_大会名）
- `tournament_plans/` — 大会出場の計画・戦略
- `books/` — ランニング関連書籍の読書メモ・要約（**gitignore対象**: 著作権のある原文テキストを含むため）
- `tools/` — データ変換などの補助ツール

## 注意事項

- `books/` ディレクトリはローカルのみ。著作権のある書籍テキストが含まれるためリポジトリには含めない。
- トレーニング計画はダニエルズのランニング・フォーミュラの理論をベースにしている。
- 言語は日本語を基本とする。
